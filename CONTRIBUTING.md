# Contributing to jcXproxy

We welcome all contributions, big or small! Whether you're fixing a bug, adding a new LLM provider, or improving documentation, your help is appreciated.

## How to Contribute

1. **Open an Issue:** Before starting a major feature, please open an issue to discuss it.
2. **Fork the Repo:** Fork `jcXproxy` to your own account.
3. **Branching:** Create a descriptive branch (e.g., `feat/add-new-provider` or `fix/typo-in-docs`).
4. **Test:** Run `npm test` locally to ensure everything works as expected.
5. **Submit a Pull Request:** Submit your PR against the `main` branch. 

Thanks for being part of the `jcXproxy` community!
